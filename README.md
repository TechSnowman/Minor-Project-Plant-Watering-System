# Minor-Project-Plant-Watering

This project is an automatic plant watering system designed and built during my first year of university.

The system monitors soil moisture levels using a moisture sensor and automatically controls a water pump. When the moisture level in the soil falls below a predefined threshold, the pump activates to water the plant. Once the soil moisture reaches the desired level, the pump automatically turns off.
